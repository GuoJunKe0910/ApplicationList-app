# 应用列表管理器 - 功能说明

## 📱 核心功能

### 1. 应用列表展示
- 显示所有已安装的应用（包括系统应用和用户应用）
- 按安装时间降序排序（默认）
- 显示应用图标、名称、包名、安装时间

### 2. 搜索功能 🔍
- 实时搜索应用名称、包名
- 支持拼音搜索（例如：搜索"wx"可以找到"微信"）
- 显示搜索结果数量

### 3. 筛选功能 📊
**按安装时间筛选：**
- 最新安装在前
- 最早安装在前

**按首字母筛选：**
- A-Z 升序
- Z-A 降序

**注意：** 搜索和筛选互不干扰，但同一时间只能使用其中一个

### 4. 应用操作功能 ⚙️

#### 启动应用 ▶️
- 一键启动任何可启动的应用
- 无需Root权限

#### 强制停止 ⏹️
- **有Root权限**：直接强制停止应用，无需确认
- **无Root权限**：跳转到系统设置页面手动停止
- 显示执行进度

#### 应用信息 ℹ️
- 跳转到系统设置中的应用详情页
- 可查看权限、存储、流量等详细信息
- 无需Root权限

#### 清除数据 🗑️
- **有Root权限**：直接清除应用所有数据（不可恢复）
- **无Root权限**：跳转到系统设置页面手动清除
- 操作前会弹出确认对话框
- 显示执行进度

#### 卸载应用 ❌
- 一键卸载用户应用
- 系统应用无法卸载（按钮会禁用）
- 无需Root权限

## 🔐 Root权限功能

### Root权限检测
应用启动时自动检测Root权限：
- ✓ **有Root权限**：显示"Root权限已获取，可使用高级功能"
- ⚠ **无Root权限**：显示"无Root权限，部分功能受限"

### Root功能列表
1. **强制停止应用** - 直接执行，无需跳转系统设置
2. **清除应用数据** - 直接执行，无需跳转系统设置

### Root命令说明
- `am force-stop <包名>` - 强制停止应用
- `pm clear <包名>` - 清除应用数据

## 🛡️ 权限说明

### 必需权限
```xml
<uses-permission android:name="android.permission.REQUEST_DELETE_PACKAGES" />
```
用于卸载应用

### 查询权限（Android 11+）
```xml
<queries>
    <intent>
        <action android:name="android.intent.action.MAIN" />
    </intent>
</queries>
```
用于查询已安装的应用列表

## 💡 使用技巧

1. **快速搜索**：支持中文、英文、拼音混合搜索
2. **批量操作**：可以结合筛选功能快速找到特定应用
3. **Root用户**：强制停止和清除数据功能可直接执行，大幅提升效率
4. **非Root用户**：所有基础功能仍可使用，高级功能会自动跳转到系统设置

## ⚠️ 注意事项

1. **Root权限**：需要设备已Root且授予本应用超级用户权限
2. **系统应用**：系统应用无法卸载，但可以停止和清除数据
3. **数据清除**：清除数据操作不可恢复，请谨慎操作
4. **兼容性**：支持Android 7.0 (API 24) 及以上版本
